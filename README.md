# trace

本项目是一个基于 JDK 17、Spring Boot 3、MySQL 和 Redis 的后台管理系统。 

## 功能特性

- 用户管理：包括用户的增删改查、权限管理等功能。 
- 角色管理：包括角色的增删改查、权限分配等功能。 
- 权限管理：包括权限的增删改查、权限分配等功能。 

## 技术栈

- JDK 17 
- Spring Boot 3 
- Spring MVC 
- Spring Data JPA 
- MySQL 
- Redis 

## 环境要求

- JDK 17 
- Maven 
- MySQL 
- Redis
